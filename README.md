## 🚀 Proyecto Destacado

### 💥 UGMM – Ring Blasting

> *Software de diseño y simulación de tronadura de abanicos para minería subterránea*

Herramienta desarrollada en Python para asistir el proceso completo de diseño y evaluación de tronaduras en abanicos. Incluye:

- 🔩 Generación automática de **perforaciones y esquemas de carga**
- ⚡ Simulación de **distribución de energía** y **contornos de daño**
- 🧱 Estimación de **fragmentación** mediante modelos empíricos e **IA**
- 📐 Cálculo de **sobreexcavación** y **costos operacionales**

`Python` · `PyQt` · `AI` · `Minería Subterránea` · `Tronadura`
